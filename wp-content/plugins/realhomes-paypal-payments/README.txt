== Description ==

This plugin works only with [RealHomes WordPress real estate theme.](https://realhomes.io/)

This plugin allows RealHomes theme website admin to add PayPal payments functionality for individual properties submitted by website users.

Plugin includes translation files for Spanish, Italian, French, German, Turkish and Portuguese languages.

== Installation ==

This section describes how to install the plugin and get it working.

1. You must have RealHomes theme properly installed on your website with required plugins.
1. Upload `realhomes-paypal-payments` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the *Dashboard > Easy Real Estate > PayPal Settings* page for the configuration.
