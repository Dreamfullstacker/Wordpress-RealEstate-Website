Provides multiple currencies support and currency switching functionality for RealHomes theme.

== Description ==

This plugin works only with [RealHomes WordPress Real Estate Theme](https://realhomes.io/).

Provides multiple currencies support and currency switching functionality for RealHomes theme.

Plugin includes translation files for Spanish, Italian, French, German, Turkish and Portuguese languages.

== Installation ==

This section describes how to install the plugin and get it working.

1. You must have RealHomes theme properly installed on your website with required plugins.
1. Upload `realhomes-currency-switcher` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the *Dashboard > Easy Real Estate > Currencies Settings* page for the configuration.

== Frequently Asked Questions ==

= Can this plugin work standalone? =

No, this plugin works only with [RealHomes WordPress real estate theme.](https://realhomes.io/)
